
##############
###  INFO  ###
##############

Every entities and values are represented on 2 BYTES.

Entities can be physical as well as virtual.

Thus, theoretically the game can have 65536 entities.

Signed integer values in [−32,767, +32,767] range.
Unsigned integer values in [0, 65,535] range.
Values can have 16 flags.
Mixed structures are allowed.

LSB (least significant bit) 0 bit numbering: FEDCBA98.76543210

Low 12 bits (BA9876543210) contains the Entity ID.

High 4 bits (FEDC) represents:
    - if 0000 then the entity itself which you can think of as
      item and/or location.
    - otherwise an entity variable

Every change in the game are represented by 2 x 2 = 4 BYTES.

Movements: entity (2) to entity (2)
New value: entity variable (2) is given a new value (2)



##################
###  ENTITIES  ###
##################

Bits BA represent the entity's main type:
    - 00 : non-card
    - 01 : other card
    - 10 : red card
    - 11 : black card



=== non-card ===
================

Bit 9 represent:
    - 0 : virtual
    - 1 : physical

-- Province --

= FORMAT:

00010100.xxxxxxxx : CONST Flags
    - Bit 0==1 if Frontier
00100100.xxxxxxxx : CONST Income
    - Bits FED : UINT Personal Income roll dice count + 4
    - Bits CBA98 : UINT Personal Income bonus + 7
    - Bits 765 : UINT State Income roll dice count + 4
    - Bits 43210 : UINT State Income bonus + 7
00110100.xxxxxxxx : CONST
    - Bits 76543210 : CONST UINT State Taxes
01000100.xxxxxxxx : CONST Base Strength
    - Bits FEDCBA98 : UINT Base Land Strength
    - Bits 76543210 : UINT Base Fleet Strength
01010100.xxxxxxxx : CONST Maximum Forces
    - Bits FEDCBA98 : UINT Maximum Armies
    - Bits 76543210 : UINT Maximum Fleets
10010100.xxxxxxxx : Flags
    - Bit 0==1 if Revolt
11100100.xxxxxxxx : Governor
    - Bits FEDC : UINT Term
    - Bits BA98.76543210 : Governor Entity ID
10110100.xxxxxxxx : Allied with War
    - Bits BA98.76543210 : War Entity ID
11010100.xxxxxxxx : Forces
    - Bits FEDCBA98 : UINT Armies
    - Bits 76543210 : UINT Fleets
11110100.xxxxxxxx : Rebel Reinforcement
    - Bits FEDC==0x1 if reinforcement is _in transit_
    - Bits FEDC==0xA if reinforcement has been _added_ to the main rebel army
    - Bits BA98.76543210 : Rebel Entity ID

= FORMAT:

00011100.00xxxxxx : CONST Flags & Era
    - Bits EDC : Era
    - Bit B==1 if Disaster 1 is Active
    - Bit A==1 if Disaster 2 is Active
    - Bit 9==1 if Standoff 1 is Active
    - Bit 8==1 if Standoff 2 is Active
    - Bit 0==1 if the Senate can vote to move this War to Inactive status after it has been attacked (and not defeated) if there is no Army currently assigned to it.
00101100.00xxxxxx : CONST Strength
    - Bits FEDCBA98 : Land Strength
    - Bits 7654 : Fleet Support
    - Bits 3210 : Fleet Strength
00111100.00xxxxxx : CONST Active Condition
    - Bits FEDC: Active Condition Key
        - FEDC==1111 : Active
        - FEDC==0100 : Active if NONE of the Entities is Rome's Province
        - FEDC==0101 : Active if ANY Entity is Rome's Province
        - FEDC==0110 : Active if ALL Entities are Rome's Province
    - Bits BA9876: Province 2 Bits 543210
    - Bits 543210: Province 1 Bits 543210
01001100.00xxxxxx : CONST Drought Condition
    - Bits FEDC: Drought Condition Key
        - FEDC==1111 : Drought
        - FEDC==0100 : Drought if NONE of the Entities is Rome's Province
        - FEDC==0101 : Drought if ANY Entity is Rome's Province
        - FEDC==0110 : Drought if ALL Entities are Rome's Province
    - Bits BA9876: Province 2 Bits 543210
    - Bits 543210: Province 1 Bits 543210
01011100.00xxxxxx : CONST Revolt Condition

        - FEDC==1111 : Revolt
        - FEDC==0100 : Revolt if NONE of the Entities is Rome's Province
        - FEDC==0101 : Revolt if ANY Entity is Rome's Province
        - FEDC==0110 : Revolt if ALL Entities are Rome's Province
    - Bits BA9876: Province 2 Bits 543210
    - Bits 543210: Province 1 Bits 543210
01101100.00xxxxxx : CONST Cost / Spoils
    - Bits FEDCBA98 : Cost
    - Bits 76543210 : Spoils

    - Bits E==1 if Attacks if Province 1 is Undeveloped

= FORMAT:

00011101.00xxxxxx : CONST Flags & Era
    - Bits EDC : Era
    - Bit B==1 if Disaster 1 is Active
    - Bit A==1 if Disaster 2 is Active
    - Bit 9==1 if Standoff 1 is Active
    - Bit 8==1 if Standoff 2 is Active
    - Bit 1==1 if Pontifex Maximus cannot be elected until this Leader is discarded.
    - Bit 0==1 ALWAYS! This makes Leader cards in par with Intrigue Leaders (Cleopatra VII)
00101101.00xxxxxx : CONST INT Strength
10011101.00xxxxxx : CONST Disaster / Standoff
    - Bits FEDC : Disaster 1 Value - 3
    - Bits BA98 : Disaster 2 Value - 3
    - Bits 7654 : Standoff 1 Value - 3
    - Bits 3210 : Standoff 2 Value - 3
10101101.00xxxxxx : CONST Matching War 1 & 2
    - Bits DCBA98 : Matching War 1 Entity ID
    - Bits 543210 : Matching War 2 Entity ID
10111101.00xxxxxx : CONST Matching War 3 & 4
    - Bits DCBA98 : Matching War 3 Entity ID
    - Bits 543210 : Matching War 4 Entity ID
11001101.00xxxxxx : CONST Destroys / Allows Concession 1 & 2
    - Bits FE==11 if Destroys Concession 1
    - Bits FE==10 if Destroys Tax Farmer Concession 1
    - Bits FE==01 if Allows Concession 1
    - Bits DCBA98 : Destroys Concession 1 Entity ID
    - Bits 76==11 if Destroys Concession 2
    - Bits 76==10 if Destroys Tax Farmer Concession 2
    - Bits 76==01 if Allows Concession 2
    - Bits 543210 : Destroys Concession 2 Entity ID
